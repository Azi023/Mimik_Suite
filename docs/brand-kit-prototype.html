Colour Palette:

Clinical Plum carries the medical authority — deep enough to feel serious, warm enough to never feel corporate. Porcelain is the clinic itself: open, clean, load-bearing whitespace. Plum Veil softens the system for panels and secondary moments, and Soft Lilac (pending confirmation) will add the single note of gentleness the treatments deserve.

Visual Tone:

Real photography only — credible skin, real light, no illustration. High-key, airy frames with one decisive plum element. Copy always sits on a translucent panel, never fights the photograph. If a layout feels busy, it is wrong.

How it aligns with the Personality:

The restraint is the personality. Generous margins and a single calm accent read exactly like the consultant who doesn't need to shout — the composition performs the credibility the copy claims.

Uniqueness vs Competitors:

The category default is loud: gold foil, heavy filters, before/after grids. Glo2Go owns the opposite corner — editorial calm and education-led frames — so the feed is recognisable at thumbnail size without a logo.
